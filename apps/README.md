# Applications

User-space applications to run on the Nomad Core device.

## Structure

This directory will contain:

- Custom UI/launcher
- Utility applications
- Games and tools
- Configuration managers

## Development

Applications should be:
- Lightweight (optimized for Pi Zero 2W)
- Touch/gamepad friendly
- Low power consumption aware
- Modular and swappable
