# Nomad Core - Planning & Documentation

Design documentation and specifications for the Nomad Core wearable computer platform.

## Current Documentation

- **[hardware/DESIGN.md](hardware/DESIGN.md)** - Complete design philosophy, system architecture, hardware components, and key dimensions

## Project Overview

The Nomad Core is a modular forearm-mounted wearable computer platform with three modules:

1. **Compute Module** - Raspberry Pi Zero 2W + UPS + batteries
2. **Display Module** - 4.3" angled screen
3. **Input Module** - Swappable gamepad/keyboard

All CAD files are in the [`cad/`](../cad/) directory. Latest STL files for printing are in [`stl/latest/`](../stl/latest/).
